# papercirclee Profile README Setup

GitHub only renders a profile README from a public repository with the exact same name as the account: `papercirclee/papercirclee`.

## Files

Place these files in that repository:

| Source in this kit | Destination in `papercirclee/papercirclee` |
|---|---|
| `README.md` | `README.md` |
| `.github/workflows/snake.yml` | `.github/workflows/snake.yml` |

## Activation

1. Create the public repository `papercirclee/papercirclee` if it does not exist.
2. Commit the two files above.
3. Open the repository's **Actions** tab and enable workflows if GitHub asks.
4. Run **Generate contribution snake** once with `workflow_dispatch`.
5. In GitHub profile settings, enable private contribution display if you want private commits counted in the contribution graph.

The snake image uses the `output` branch generated by the workflow:

```md
https://raw.githubusercontent.com/papercirclee/papercirclee/output/github-contribution-grid-snake-dark.svg
```

If the image is blank at first, the workflow likely has not created the `output` branch yet.
